== Description ==

With ValidateCertify, guarantee the authenticity and integrity of the certificates issued, providing confidence to both your students and those who verify them. Simplify the validation process and improve your users' experience quickly and easily, without having to change platforms or websites.

Setting up validation of your certificates is simple: simply upload your certificate database to our plugin. This database includes crucial information such as names, surnames, course, date and certificate code. **Names, Last name, Course, Date and Certificate Code.**


###Features of ValidateCertify - Free###


- Manually add certificates to your database.
- Edit existing certificates with the certificate code.
- Delete any records using the certificate code.
- Easily view the database of your certificates.
- Use a ShortCode to integrate the validation option on any page.
- Shows the following data: **First name, Last name, Course, Date and Certificate Code.**


##Elevate your Experience with ValidateCertify Pro##


###Enhanced Features of [ValidateCertify - Pro](https://systenjrh.com/plugin-validatecertify/)###

- Add certificates in bulk, up to 1000 certificates in a single upload.
- Automatically detects duplicate codes in bulk upload, uploading only new records.
- Export your database at any time for greater flexibility.
- Remove the entire base if you need to.
- Customize the ShortCode button according to your preferences.
- Integrate the Elementor ValidateCertify widget for an even smoother experience.
- Shows the following data: **First name, Last name, Course, Date and Certificate Code.**


##Take your Validation to the Next Level with ValidateCertify ProPlus##


###Advanced Features of [ValidateCertify - ProPlus](https://systenjrh.com/plugin-validatecertify/)###


- Automatically generate certificate codes from auto-upload, further simplifying the process.
- In addition to the standard data, displays the "certificate" field, formerly "Courses", for greater specificity.
- Take advantage of all the features of ValidateCertify Pro, along with these new improved features.

== Installation ==

**From your WordPress dashboard**
1. Install the plugin through the WordPress plugins screen directly.
2. Activate ValidateCertify from your Plugins page.

**From WordPress.org**
1. Search and download ValidateCertify
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

**How to add shortcode to verify certificates**

**How many fields do you have**

Have five parameters, Name, Last name, course, date and certificate code.
